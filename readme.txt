To use your extensions with SmartDashboard...

 - Run the SmartDashboard Standalone/Vision Installer (instructions: http://wpilib.screenstepslive.com/s/3120/m/7932/l/93058-smart-dashboard-standalone-vision-installer)
 - Open and Build this project in NetBeans
 - Copy the JAR file generated by NetBeans from the "[project_root]\dist\" directory into the "C:\Program Files\SmartDashboard\extensions\" directory
 - Run the FRC Driver Station
